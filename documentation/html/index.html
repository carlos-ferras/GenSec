Sequence-ToolKit

The automated luminescence reader LF02 has been designed to carry out complex sequences of beta irradiation and luminescence measurement on a considerable number of samples without exposing them to light. The conditions in which these sequences are performed, are determined by a significant number of parameters.

Sequence-Toolkit (STK) is used in the generation and posterior analysis of the measurement sequences. The STK was programmed on the basis of technical task elaborated by the LF02 developers.

For the accomplishment of these tasks the STK has a set of applications (GenSec, GenRep, GenVis) and a help assistant .

  • Qt Designer: To create user interfaces.
  • Inkscape and GIMP: For iconography design.

Cross-Platform

These applications can be used on any operating system without making changes to the source code, for this goal you just have to make a compatible installer with the desired system.

Multilanguage

The default package language is English but it can be changed to every possible language, just adding a file with texts translations, it has incorporated some languages by default. When you open any application of this package, it recognize the operating system language, and if there is a translation available in this language, it will be opened with this language, otherwise it will be opened with the default language.
